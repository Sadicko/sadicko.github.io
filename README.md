# sadickodai.dev — Personal Portfolio

Personal portfolio website for **Sadick Sessah-Odai** — Senior Software Engineer, Full-Stack Developer, and Founder of [INCREGHANA](https://increghana.com). Based in Accra, Ghana.

## Live Site

> Hosted on GitHub Pages · Custom domain `sadickodai.dev` (coming soon)

## Stack

Pure HTML, CSS, and vanilla JavaScript — no build tools, no frameworks, no dependencies beyond:

- [Lucide Icons](https://lucide.dev) (CDN)
- [Google Fonts](https://fonts.google.com) — Playfair Display + Cabinet Grotesk
- [Formspree](https://formspree.io) — contact form backend

## Features

- Custom animated cursor
- Scroll-reveal animations
- Responsive layout (mobile + desktop)
- SEO optimised — JSON-LD structured data, Open Graph, Twitter Card
- Favicon set + PWA web manifest

## Deployment

This site is deployed via **GitHub Pages** from the `main` branch.

To deploy your own fork:
1. Push to a repo named `yourusername.github.io` for a root domain, or any repo name for a project subdomain
2. Go to **Settings → Pages → Source → main / root**
3. Once you have a custom domain, add it under **Settings → Pages → Custom domain** and update the `canonical`, `og:url`, `og:image`, and `twitter:image` URLs in `index.html`
